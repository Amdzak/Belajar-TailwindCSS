sebelum memakai tailwindcss kita harus terlebih dahulu untuk melakukan setup dan konfigurasi terlebih dahulu

    SPACING & SIZING:
    sistem penulisan margin hampir mirip dengan penulisan bootstrap yaitu 
    m-0 = untuk margin ke 4 sisinya
    mx-0 = untuk sumbu x (kanan kiri)
    my-0 = untuk sumbu y (atas bawah)
    mt-0 = untuk sisi atas
    mr-0 = untuk sisi kanan
    mb-0 = untuk sisi bawah
    ml-0 = untuk sisi kiri
    
    jika ada beberapa pengguanan tailwindcss yang tidak sesuai dengan keinginan maka kita dapat melakukan dengan kustomisasai sendiri dapat di lihat di web dokumentasi tailwindcss pemanggilan tinggal tulis aja nama custom nya tadi
    sistem penulisan di ata berlaku untuk
    PADDING = p
    WIDTH   = w
    HEIGHT  = h    

    TYPOGRAPHY :
    berfungsi untuk mengatur text dan font, dengan menuliskan jenis font maka bisa menggunakan font seris,sanserif
    font-serif
    font-sanserif
    jika ingin menggunakan custom font di tailwind kita harus memiliki nama fontnya terlebihdahulu bisa dengan import dari font google
    lalu kita buat customisasi sama seperti customisasi pada pengguanan margin
